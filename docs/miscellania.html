Miscellaneous Functions

Blink Size

Blink size has a default of 1 pixel but this can be increased to 2x2, 3x3, etc. with F10 and decreased with F11. Excluded blinks have a default size of 3x3 pixels to make them easier to see. This can be increased with Alt + F10 and decreased with Alt + F11.

Adding a scale bar

To add a scale bar to the press F5 which brings up a box allowing the user to enter the length of the scale bar in nanometers. Position the cursor to where the left bottom corner of the scale bar is to be placed and press the center mouse button. If the scale bar is in the wrong position pressing F5 removes it and press F5 again to restart the process.

Distance Measurement

Distances can be measured using the l key. After pressing the key and entering measurement mode, left click on mouse at the chosen start position in the image and drag the cursor to the finish point. The distance is updated as the cursor is moved and is fixed once the left mouse button is released. Further measurements can be made by clicking on a new point and then dragging the cursor to the target. Press l to exit the measurement mode.

Saving a subset of the data

A subset of the data can be saved by first drawing an ROI. Press the F3 and the left click on the starting position of the ROI dragging the cursor until the ROI covers the area of interest. On release of the left mouse button a window will pop up with a suggested file name (chosen so as not to overwrite existing files) as well the dimensions of the area. Pressing OK on the pop-up will save an 3dSTM file with the given name and blinks within the ROI.
