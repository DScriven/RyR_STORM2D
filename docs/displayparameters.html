Display Parameters

All of these parameters can be changed to alter image that appears on the screen. They can be changed and then entered using the "Modify" key and require minimal extra computation.

Cluster Boundaries

The vermillion lines shows the boundaries of the base (i.e 30 nm) clusters. The image was generated with an alpha of 700.

Decreasing the alpha tightens the fit and follows the perimeter contour more tightly as well causing some small holes to appear, while increasing the alpha has the opposite effect.

Convex Hull

The convex hull (skyblue lines) outlines the minimum polygon that will envelop a cluster group. It is the best way to visually determine the boundaries of the cluster groups as the neighbourhood limit (NL) is increased. The hulls represent a NL of 50 nm.

Minimum Ellipse

The minimum ellipse (bluish-green lines) allows an estimate in the changes in shape that occur on phosphorylation with isoproterenol. The minimum ellipse always contains the convex hull. The ellipses represent a NL of 50 nm.

Excluded Blinks

The purple blinks represent those excluded during the fitting process as having too few blinks or being too small to be a tetramer. The actual size is smaller than shown here; the blink size has been increased to 3x3 pixels ( Alt + F10) so that the points can be visualized.

Log Threshold

To isolate regions of high blink density you can use the Log Threshold. All pixels below this threshold are in skyblue while those above are in vermillion - as in the image below. In this case the Log Threshold is -1.6.

This threshold is image-wide so you may have to adjust it for certain clusters.

Cluster Numbers

Displays the numbers associated with each cluster. These numbers will change as the NL changes; those shown are for an NL of 50nm.
