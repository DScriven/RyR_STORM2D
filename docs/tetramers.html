Adding Tetramers to the Blink Image

If a tetramer.dat file already exists it will be read in automatically immediately after the blink file.

Adding, modifying and classifying tetramers

Adding a tetramer

Press the a key and left click the mouse in the approximate position you want the tetramer to appear. The position of the tetramer can be moved one pixel at a time using the four arrow keys on the keyboard. The rotation is controlled by the mouse wheel. Once you are satisfied with the position press the s key to save the tetramer position. Once this is done another tetramer can be added.

Modifying a tetramer

First, identify the tetramer by pressing the m key and left click on the desired tetramer, which will change colour to black. To alter the position and orientation of the tetramer use the four arrow keys and the mouse wheel. Press the s key once finished

Delete a Tetramer

To delete a tetramer, use the m key and left click to highlight it, then use the d key and it will be erased - Warning - this cannot be undone!

Valid Add/Modify keys

EscImmediately stop addition or modification
Arrow keys
(←, →, ↑ and ↓)		Move the tetramer 1 pixel in that direction
Mouse wheelRotate tetramer
dDelete the tetramer (after m)
sSave tetramer after addition or modificatioon

Classify a Tetramer

To classify a tetramer, press the c key and left click on the desired tetramer to highlight it, then use one of the following keys to classify it.

EscImmediately stop classification
b'both'
c'checkerboard'
i'isolated'
s'side-by-side'
u'unclassified'

Once the key is pressed the tetramer is classified

Analysing the tetramer data

The tetramers do not appear by default if loaded from a file - the F7 key is a toggle that will cause the added or stored tetramers to appear or disappear.

The NND values can be displayed at any time by pressing n on the keyboard, but a pixel size of ~1nm is needed to clearly see the values. The NND value will only appear if there are two or more tetramers in a cluster. The blink image can be hidden with the F6 after which the tetramers will be shown in outline on a black background with the NNDs if n has been pressed. Both n and F6 are toggles whose effects can be reversed by pressing them again. (The NNDs will no longer appear, the blinks will reappear, and the tetramers will be represented by squares).

Although the default tetramer width is 27 nm it is possible to change that width by pressing the F8 key

The tetramer and NND data is saved in two files: fn_tetramer.dat and fn_nnd.dat (where fn is the 3dSTM filename). The first file stores the (x,y) position of each tetramer, its orientation and cluster number while the second is a list of the calculated NND files. These files can be saved manually at any time by using the F9 key, although finishing an analysis pressing any one of the following keys or icons: Esc, Close or the file browser button will automatically calculate the NNDs and save the two files.
